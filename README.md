# 🍳 Kitchen Quest

App de meal prep y alimentación saludable con planificación semanal,
cálculo nutricional y listas de mercado automáticas.

## Estado del proyecto
🚧 En desarrollo activo — Mundo 1: Cimientos

## Stack tecnológico
- **Frontend:** HTML · CSS · JavaScript → React
- **Backend:** Python · FastAPI
- **Base de datos:** PostgreSQL
- **Despliegue:** Vercel + Railway

## Sobre este proyecto
Desarrollado como proyecto de aprendizaje y portfolio por Yampis,
arquitecto en transición al desarrollo de software.
